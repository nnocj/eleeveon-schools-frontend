# Branch Admin Calendar, Timetable, Announcements and Messages Modules

Drop these files into `app/branch-admin/modules/`.

Files:
- Calendar.tsx
- BranchTimetable.tsx
- ClassTimetable.tsx
- TeacherTimetable.tsx
- ExamTimetable.tsx
- ResourceTimetable.tsx
- Announcements.tsx
- Messages.tsx

They assume your `db.ts` has calendar/schedule/message/announcement tables and your `app/lib/calendar` + `app/lib/scheduling` packages are already installed.
